## 🧩 Framework Features

### ✔ Request & Response Specifications  
Centralized configuration for:
- Base URI  
- Base Path  
- Headers  
- Authentication  
- Logging  

### ✔ Reusable API Methods  
Supports:
- `GET`
- `POST`
- `PUT`
- `DELETE`
- `PATCH`

### ✔ TestNG Integration  
- Test grouping  
- Assertions  
- Parallel execution support  

### ✔ JSON Payload Handling  
- External JSON files  
- POJO serialization/deserialization  

### ✔ Environment Support  
Switch between environments using:
- `config.properties`
- Maven profiles  
- System variables  

### ✔ Logging & Reporting  
- Request/response logs  
- TestNG HTML reports  

---

## 🧪 Example Test Flow

1. Build request specification  
2. Add headers / params  
3. Send request  
4. Validate:
   - Status code  
   - Response body  
   - Headers  
   - JSON schema (optional)
